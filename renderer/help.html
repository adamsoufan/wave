Help Guide

Welcome to Wave

Wave is a powerful desktop application that allows you to control your computer using hand gestures detected through your webcam. This guide will help you understand how to set up macros, create gesture mappings, and use the application effectively.

 

Getting Started

The Wave application consists of three main components:

1 Macro Hub: This is where you create sequences of actions (macros) that will be executed when a gesture is detected.
2 Mapping Hub: Here you can link specific hand gestures to the macros you've created.
3 Home: The main screen where you can enable gesture detection and see all your active mappings.
Pro Tip: Before starting, make sure your webcam is properly connected and has a clear view of your hands.

 

Step-by-Step Tutorial

Creating a Macro

  1. Go to the Macro Hub page using the sidebar.
  2. Click the New Macro button in the top right corner.
  3. Add a descriptive name for your macro in the "Name" field.
  4. Add one or more actions to your macro from the available types:
    • Key Press: Simulate keyboard input (e.g., Ctrl + C for copy).
    • Upload Script: Run a custom script or application.
    • Command Line: Execute a command-line command.
  5. Arrange your actions in the desired order using drag and drop.
  6. Click Save to store your macro.

 

Mapping Gestures

  1. Navigate to the Mapping Hub page.
  2. Click New Mapping to create a new gesture mapping.
  3. Select the macro you want to trigger from the dropdown menu.
  4. Give your mapping a descriptive name.
  5. Click on either the left or right hand block (or both) to select a gesture.
  6. Choose the gesture(s) you want to use from the popup menu.
  7. Click Save to create the mapping.

 

Using Gesture Detection

  1. Go to the Home page.
  2. Click the Start Detecting button to enable gesture recognition.
  3. Position yourself in front of your webcam.
  4. Perform the gestures you've mapped to trigger the associated macros.
  5. To stop detection, click the Stop Detecting button.

 

Practical Example: Media Controls

Scenario: Control music playback with hand gestures

Let's create a set of gesture controls for your media player:

 

Step 1: Create the Play/Pause Macro

  1. Go to Macro Hub and click New Macro.
  2. Name it "Play/Pause".
  3. Add a Key Press action and set it to "Space" (the universal play/pause shortcut).
  4. Click Save.

 

Step 2: Create Volume Up/Down Macros

  1. Create a new macro named "Volume Up".
  2. Add a Key Press action and set it to "Ctrl + Up Arrow".
  3. Save and create another macro named "Volume Down".
  4. Add a Key Press action and set it to "Ctrl + Down Arrow".

 

Step 3: Map Gestures

  1. Go to Mapping Hub and create a new mapping.
  2. Select the "Play/Pause" macro and name the mapping "Play/Pause".
  3. For the right hand, select the "Thumbs Up" gesture (üëç).
  4. Save the mapping.
  5. Create a second mapping for "Volume Up" with the "Point Up" (üëÜ) gesture.
  6. Create a third mapping for "Volume Down" with the "Point Down" (üëá) gesture.

 

Step 4: Use Your Gestures

  1. Return to the Home page and click Start Detecting.
  2. Open your media player and start playing something.
  3. Make a thumbs up gesture (üëç) to pause/play the music.
  4. Use point up (üëÜ) to increase volume.
  5. Use point down (üëá) to decrease volume.

 

Tips for Better Detection

  • Ensure good lighting in your workspace for better hand detection.
  • Keep your hand within the webcam's field of view.
  • Make clear, deliberate gestures for more reliable detection.
  • Try to avoid busy or cluttered backgrounds that might confuse the detection system.
  • For two-handed gestures, make sure both hands are visible to the camera.

 

Troubleshooting

Gesture Not Being Detected

  • Ensure your webcam is working properly.
  • Check that you've clicked "Start Detecting" on the Home page.
  • Position your hand clearly in front of the camera.
  • Try making the gesture more deliberately or with better lighting.

 

Macro Not Executing

  • Verify that the mapping is enabled (toggle should be "ON" on the Home page).
  • Test the macro directly using the "Test" button in the Macro Hub.
  • Ensure the application you're trying to control is the active window.
